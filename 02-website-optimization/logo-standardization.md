# Logo Standardization

## Purpose

This document records the logo update completed for the Potere Search website.

---

## Objective

Update the Potere Search logo by removing the tagline beneath the logo mark to create a cleaner and more consistent brand appearance across the website.

---

## Change Made

The tagline was removed from the Potere Search logo.

The updated logo now includes only:

- PS magnifying glass mark
- Potere Search wordmark

---

## Reason for the Update

The previous logo included a tagline beneath the main logo. Removing the tagline makes the logo cleaner, easier to read, and more suitable for use in the website header and footer.

---

## Implementation

The updated logo was uploaded through the WordPress Customizer.

Path:

Appearance → Customize → Header → Site Title & Logo

The footer logo updated automatically because it uses the same global logo asset as the header.

---

## Areas Updated

- Header logo
- Sticky header logo
- Footer logo

---

## Verification

The updated logo was checked on the website header and footer to confirm that the change applied consistently.
