# Loxo ATS Migration

## Project Overview

One of the first major growth and operational initiatives was evaluating and planning the migration of recruiting operations from RecruiterFlow to Loxo ATS.

## Business Objective

Create a streamlined recruiting infrastructure while ensuring:

- Public jobs appear on the company website
- Confidential searches remain private
- Search Directors maintain publishing control
- Job posting workflows become easier to manage

## Discovery Process

### Existing ATS

RecruiterFlow

### Existing Website Integration

The website Job Postings page was connected to RecruiterFlow through an embedded iframe located within Elementor.

### Investigation Activities

- Reviewed WordPress configuration
- Audited Job Postings page
- Investigated Elementor implementation
- Identified RecruiterFlow integration method
- Reviewed Loxo Careers Pages functionality
- Confirmed publishing controls for confidential searches

## Key Findings

### RecruiterFlow

Jobs were displayed through an iframe embedded directly into the website.

### Loxo

Loxo Careers Pages allows jobs to be published selectively based on recruiter controls.

### Confidential Searches

Certain executive search assignments should not be publicly displayed and require controlled publishing.

## Planned Outcome

Replace RecruiterFlow job display functionality with Loxo Careers Pages while preserving confidentiality workflows and improving operational efficiency.
