# Loxo ATS Migration

## Overview

This project migrated the Potere Search Job Postings page from RecruitFlow to Loxo.

The previous RecruitFlow integration contained outdated job listings and required manual maintenance. The new implementation connects the website directly to the Loxo Careers Page, allowing published jobs to automatically appear on the website.

## Goals

* Replace RecruitFlow with Loxo
* Eliminate outdated job listings
* Create a live connection between the website and Loxo
* Improve candidate experience
* Reduce manual website maintenance

## Deliverables

* RecruitFlow removed
* Loxo Careers Page implemented
* Live job synchronization established
* Mobile blocking issue resolved
* Documentation completed
