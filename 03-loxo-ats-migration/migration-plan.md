# Migration Plan

## Objective

Replace the existing RecruitFlow job board with the Loxo Careers Page and establish Loxo as the single source of truth for public job postings.

## Phase 1 - Discovery

Completed

* Reviewed current RecruitFlow implementation
* Reviewed Loxo Careers Page functionality
* Identified outdated jobs on website
* Confirmed migration approach

## Phase 2 - Implementation

Completed

* Removed RecruitFlow integration
* Added Loxo Careers Page iframe
* Published jobs through Loxo
* Verified job synchronization

## Phase 3 - Troubleshooting

Completed

Issue identified:

* Social sharing toolbar blocked job content on mobile devices

Resolution:

* Contacted Loxo Support
* Implemented disable_addthis parameter

## Phase 4 - Launch

Completed

* Published updated Job Postings page
* Verified live jobs display correctly
* Confirmed future jobs will be managed through Loxo

## Result

The Potere Search website now displays live jobs directly from Loxo.
